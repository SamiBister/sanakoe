## Aloittaminen

### Sovelluksen avaaminen

1. Avaa selain (Chrome, Firefox tai Safari)
2. Mene sovelluksen osoitteeseen
3. Näet tervetulonäytön suomeksi

### Kielen vaihtaminen

Klikkaa oikeassa yläkulmassa olevaa kielipainiketta:

- 🇫🇮 **Suomi**
- 🇬🇧 **English** (englanti)

## Sanojen lisääminen

Voit lisätä sanoja kahdella tavalla:

### Tapa 1: Lataa CSV-tiedosto 📁

Sopii, jos opettaja on antanut sanalistan tiedostona!

1. Klikkaa **"Lataa CSV"**
2. Klikkaa **"Selaa tiedostoja"** tai raahaa tiedosto laatikkoon
3. Sovellus hyväksyy `.csv` ja `.txt` tiedostot
4. Näet esikatselun sanoistasi
5. Sanamäärä näkyy alhaalla

**Tiedoston muoto:**

```
koira,dog
kissa,cat
talo,house
```

Sekä pilkku (`,`) että puolipiste (`;`) toimivat erottimena!

### Tapa 2: Kirjoita sanat käsin ✏️

Sopii, kun haluat tehdä oman harjoituslistan!

1. Klikkaa **"Syötä Käsin"**
2. Kirjoita sanat taulukkoon:
   - **Vasen sarake (Sana):** Kysymyksenä näkyvä sana
   - **Oikea sarake (Käännös):** Oikea vastaus
3. Paina **Tab** siirtyäksesi seuraavaan soluun
4. Uusia rivejä tulee automaattisesti lisää
5. Laskuri näyttää valmiiden sanaparien määrän

**Vinkkejä:**

- Molemmissa sarakkeissa pitää olla tekstiä, jotta sanapari lasketaan
- Merkinnät tallentuvat automaattisesti!
- Klikkaa **"Tyhjennä Kaikki"** aloittaaksesi alusta (kysyy varmistuksen)

## Kokeen tekeminen

### Aloittaminen

1. Kun sanat on lisätty, klikkaa isoa **"🚀 Aloita Koe"** -painiketta
2. Koe alkaa heti ja ajastin käynnistyy!

### Vastaaminen

1. Näet sanan kortin yläosassa
2. Kirjoita vastauksesi tekstikenttään
3. Paina **Enter** tai klikkaa **"Tarkista"**

### Oikea vastaus ✅

- Näet kannustavan viestin kuten "Hienoa!"
- Seuraava sana tulee automaattisesti sekunnin kuluttua
- Tähti näyttää onnistumisesi!

### Väärä vastaus → Harjoitustila 📝

Jos vastaat väärin:

1. Näet oikean vastauksen keltaisella pohjalla
2. **Harjoitustila** alkaa - kirjoita oikea vastaus 3 kertaa
3. Laskuri näyttää edistymisesi (1/3, 2/3, 3/3)
4. Kolmen oikean toiston jälkeen palaat kokeeseen
5. Sana tulee myöhemmin uudelleen vastattavaksi

**Huom:** Harjoitustila auttaa oppimaan - se ei vaikuta pisteidesi määrään!

---

## Harjoitustila

Kun vastaat väärin, siirryt harjoitustilaan:

1. **Oikea vastaus näkyy** - kopioi se tarkasti
2. **Kirjoita 3 kertaa** - tämä auttaa muistamaan
3. **Laskuri** näyttää: 1/3 → 2/3 → 3/3
4. **Valmis!** Palaat normaaliin kokeeseen

Harjoittelun jälkeen sana tulee vielä myöhemmin uudelleen, jotta voit näyttää oppineesi sen!

## Sanalista

Klikkaa **"≡ Sanat"** -painiketta oikeassa yläkulmassa nähdäksesi kaikki sanasi!

### Tilakuvakkeet

| Kuvake | Merkitys                                     |
| ------ | -------------------------------------------- |
| ✅     | **Ratkaistu** - Vastasit oikein              |
| 🔁     | **Ratkaisematta** - Ei vielä vastattu        |
| ⚠️     | **Virhe** - Väärin ensimmäisellä yrityksellä |

### Suodattimet

- **Kaikki** - Näytä kaikki sanat
- **Ratkaisemattomat** - Sanat joita ei ole vielä vastattu
- **Virheet** - Sanat joissa teit virheen

Nykyinen sana on korostettu keltaisella reunuksella.

## Tulokset

Kun olet vastannut kaikkiin sanoihin, näet tuloksesi:

### Pisteet

- **Sanoja yhteensä:** Kuinka monta sanaa harjoittelit
- **Yrityksiä:** Kuinka monta kertaa vastasit
- **Aika:** Kuinka kauan koe kesti

### Harjoiteltavat sanat

Lista sanoista, joita et osannut ensimmäisellä yrityksellä - näitä kannattaa harjoitella lisää!

### Ennätykset 🏆

Sovellus muistaa parhaat tuloksesi jokaiselle sanalistalle!

- **Uusi ennätys!** näkyy kun voitat aiemman tuloksesi
- Ennätykset tallennetaan sekä **yrityksille** että **ajalle**
- Konfetti juhlii ennätystäsi!

### Mitä seuraavaksi?

- **"🔄 Pelaa uudelleen"** - Samat sanat, uusi järjestys
- **"🏠 Uusi lista"** - Palaa lisäämään eri sanoja

## Vinkkejä 💡

1. **Ota aikaa** - Tarkkuus on tärkeämpää kuin nopeus
2. **Käytä harjoitustilaa** - Se todella auttaa muistamaan!
3. **Tarkista sanalista** - Katso sanoja joista et ole varma
4. **Voita ennätyksesi** - Yritä käyttää vähemmän yrityksiä joka kerta
5. **Harjoittele säännöllisesti** - Lyhyet päivittäiset harjoitukset toimivat parhaiten

## Ongelmatilanteet

### "Sanani eivät tallentuneet"

- Sanat tallentuvat selaimen muistiin
- Ne säilyvät samalla laitteella ja selaimella
- Selaimen tietojen tyhjentäminen poistaa sanat

### "CSV-tiedosto ei lataudu"

- Varmista että tiedosto on `.csv` tai `.txt` muodossa
- Tarkista että jokaisella rivillä on täsmälleen 2 saraketta (sana, käännös)
- Kokeile pilkkua tai puolipistettä erottimena

### "Koe ei käynnisty"

- Varmista että sinulla on vähintään 1 valmis sanapari
- Molemmissa sarakkeissa pitää olla tekstiä

### "Näen virhesivun"

- Klikkaa **"Yritä uudelleen"**
- Klikkaa **"Palaa alkuun"** aloittaaksesi alusta
- Jos ongelma jatkuu, päivitä sivu
